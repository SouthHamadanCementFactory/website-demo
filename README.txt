نسخة موجهة للهواتف – مصنع جنوب همدان
====================================
- تصميم Mobile-first: أحجام نصوص مضغوطة، أزرار كبيرة، أعمدة مفردة، شريط إجراءات سفلي ثابت.
- التقنيات: Bootstrap RTL + Swiper + AOS + Font Awesome + Google Fonts (Tajawal).
- ملفات:
  - index.html
  - assets/css/style.css
  - assets/js/app.js
  - assets/images/ (SVG placeholders)

نصائح أداء للهاتف:
- استخدم صور WebP/JPG خفيفة بدل SVG المؤقتة.
- فعّل cache على الخادم.
- قلّل المكاتب إن رغبت بمزيد من الخفة.

تشغيل محلي:
- افتح index.html مباشرة، أو استخدم خادم محلي لمعاينة أفضل.
